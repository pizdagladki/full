# 00. Обзор продукта

**Суть:** онлайн-платформа для игры в гляделки 1-на-1. Игроки видят друг друга через веб-камеры. Проигрывает тот, кто первым моргнул. Победитель получает автоматический видео-эдит момента победы под музыкальный трек — его можно скачать и запостить в TikTok/Instagram.

**Ключевой дифференциатор:** не сама игра (она простая), а **победные эдиты + шеринг** — это вирусный двигатель роста. Человек выиграл → получил красивый ролик → выложил в соцсети → привёл новых игроков.

**Платформа:** веб (десктоп + мобильный браузер). Аудитория вероятно преимущественно мобильная.

**Режимы игры:**

- Рейтинговый (с ELO)
- Безранговый (без влияния на рейтинг)
- Пригласить друга (приватная комната)

**Аудитория:** на старте международная (платежи через Stripe), РФ — позже.
