# 08. Монетизация и удержание

> Зона: логика всех потоков (магазин, покупки, инвентарь, реклама-интеграции) — `[AGENT]`. Карточки товаров и магазин — `[AGENT-SCAFFOLD]` каркас + `[OWNER-DESIGN]` финальный вид.

Четыре потока. Единый магазин (отвлекалки + эдиты в одном сторе).

1. **Отвлекалки** (платные, тиры) — см. `07-distractions.md`. P2W, осознанный риск.
2. **Платные эдиты (косметика) ★ сильнейший поток:** 10 бесплатных + 10 платных эдитов победы. Покупка в магазине, разовая НАВСЕГДА (купил → владеешь). Честная косметика — НЕ влияет на исход, не злит проигравших, работает во всех режимах. **Синергия с шерингом:** купил красивый эдит → выиграл → пошерил в TikTok → подписчики хотят такой же → приходят и покупают.
3. **Rewarded video:** посмотрел рекламный ролик → бесплатная отвлекалка в инвентарь. Показ ДО боя, НЕ в бою. Через игровую рекламную сеть (AdMob / Unity Ads) — безопаснее и доходнее для игр.
4. **AdSense баннеры:** на всех экранах КРОМЕ батл-скрина (чистые зоны: магазин, домашний, итоги).

**На будущее (НЕ сейчас):** премиум-подписка с фишками.

## Связь с системой поинтов (см. `11-points-system.md`)

- **Двойная цена айтемов:** каждый айтем магазина (отвлекалки, эдиты) несёт ДВЕ цены — реальные деньги ИЛИ поинты (`points_price` nullable; `null` → только за деньги). Поинты зарабатываются активностью, не покупаются.
- **Деление премиум (только деньги) / базовый (можно за поинты)** — TBD (с конкретным списком айтемов). Схема магазина поддерживает обе цены изначально.
- **Защита доната холдом:** возможность холдить поинты под будущий аирдроп создаёт альтернативную стоимость трате (потратил поинты = пожертвовал долей аирдропа) → тормозит трату поинтов и подталкивает покупать за деньги. Поэтому поинты НЕ убивают денежную монетизацию, а дополняют её.

## Осознанный риск (реклама)

- AdSense насторожен к живым немодерируемым камерам незнакомцев (ассоциация с Chatroulette) → возможен бан аккаунта при 18+ в кадре рядом с рекламой. Проверить политики Google ДО внедрения.

## Удержание (retention) — без спец-механик в MVP

- Работает на органике: рейтинг/уровни (цель расти), шеринг в TikTok (возврат за реакциями), коллекция эдитов, **система поинтов** (накопление + холд под аирдроп — см. `11-points-system.md`), сам core loop (бой → эдит → ещё бой).
- Спец-механики (сезоны, winstreak, дейлики) → бэклог.
- Логика: сначала проверить, цепляет ли петля; если да — навешивать удержание. Монетизация без удержания бессмысленна.
